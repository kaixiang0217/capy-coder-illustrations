# Capybara Coder Prompt Examples

这是 Capybara Coder IP 的 prompt 使用示例集合。展示了如何在不同场景下使用 Capy Coder 进行插图生成。

## 场景一：只做配图规划

```text
Use $capy-coder-illustrations 先不要生图。
请分析下面这篇关于代码调试的文章，输出 5 张左右的 shot list。
每张图写清楚：放在哪段后、主题、核心意思、结构类型、Capy 在做什么、建议中文标注词。

[粘贴你的文章内容]
```

## 场景二：直接生成正文配图（编码主题）

```text
Use $capy-coder-illustrations 把下面这篇关于编程的文章生成 4 张 Capybara Coder 正文配图。
要求：16:9 横版、纯白背景、黑色手绘线稿、少量红橙蓝中文手写批注。
Capy 必须参与每张图的核心动作，展现"热爱编码、享受休息"的特性。

[粘贴你的文章内容]
```

## 场景三：为单个概念生成一张图（Capy 特定场景）

```text
Use $capy-coder-illustrations 为"代码写完了，终于可以睡觉了"生成一张正文配图。
画面要怪诞但清爽，Capy 必须承担核心动作——展现它完成工作后的满足感和睡意。
风格：黑白线稿、16:9 横版、大量留白。
```

## 场景四：Capy 在编码

```text
Use $capy-coder-illustrations 为"专注编码的状态"生成一张图。
Capy 应该坐在笔记本电脑前，双手放在键盘上，眼神专注但仍然冷静，表现出"虽然在 debug 但依然很镇定"的气质。
背景应该有一些混乱的代码线条来表现 bug 的存在，但 Capy 看起来完全不担心。
注释建议："这不是 bug，这是特性" 或 "保持冷静，继续编码"。
```

## 场景五：Capy 睡觉

```text
Use $capy-coder-illustrations 为"工作与休息的平衡"生成一张图。
Capy 应该睡在键盘上或靠在笔记本上，表现得很满足、很放松。
可以在睡梦中画一些代码符号或浮动的线条，表现"在梦里也在写代码"的程序员文化。
注释建议："我在梦里也在写代码" 或 "充分的睡眠是 bug fix 的一部分"。
```

## 场景六：Monday Morning（周一早上）

```text
Use $capy-coder-illustrations 为"周一早上的程序员"生成一张图。
Capy 应该拿着一杯咖啡，缓缓向笔记本电脑走去，眼神半睁半闭，身体略显懒散。
整个场景应该表现出"虽然困，但我必须开始编码"的经典程序员状态。
注释建议："我需要咖啡才能 debug" 或 "周一需要双倍咖啡因"。
```

## 场景七：功能完成庆祝

```text
Use $capy-coder-illustrations 为"功能开发完成"生成一张图。
Capy 应该满足地睡在一堆代码块或文件上面，表现出"辛苦工作后的胜利感"和"现在终于可以休息了"。
背景可以是堆积的代码块或完成状态标记。
注释建议："功能完成，现在可以睡觉了" 或 "Deploy 完成，该睡觉了"。
```

## 场景八：编辑现有的图

```text
Use $capy-coder-illustrations 帮我编辑这张图，去掉右下角的"调试中"标签，其他内容保持不变。
```

## 进阶用法：自定义 Capy 场景

如果想创建不存在的 Capy Coder 场景，可以这样描述：

```text
Use $capy-coder-illustrations 为"[你的概念或文章片段]"生成一张正文配图。

角色：Capybara Coder（Capy）——一只圆润、冷静、热爱编码和睡眠的水豚。

场景需求：
- Capy 的具体动作：[Capy 应该在做什么]
- 背景元素：[场景中还应该出现什么]
- 情感基调：[这张图应该传达什么感觉]
- 中文标注建议：[可以加什么简短的中文批注]

风格要求：
- 16:9 横版、纯白背景、黑色手绘线稿
- 大量留白（40-60% 内容，40-60% 空白）
- 少量红/橙/蓝中文手写批注
- 怪诞、清爽、有创意，但不幼稚
- Capy 必须是核心元素，不能只是装饰
```

## 关于 Capybara Coder IP 的特性

- **永远冷静**：再复杂的 bug 也不会让 Capy 惊慌失措
- **热爱编码**：Capy 和笔记本电脑形影不离
- **热爱睡眠**：Capy 善于在任何地方找到睡眠的机会
- **怪诞美学**：线条粗细有变、略微不规则，但整体和谐
- **极简主义**：纯黑白、无渐变、无阴影，只用线条表达
- **可关联性**：面向开发者和技术工作者，表现出真实的工作和休息状态
